# Gravitile

A merge puzzle for iOS with a twist you can feel: **gravity rotates 90°
after every move**, tiles tumble, and matching tiles cascade together with
multiplying scores.

| | |
|---|---|
| Platform | iOS 17+, iPhone, portrait |
| Stack | Swift 6, SwiftUI, StoreKit 2, GameKit, CoreHaptics |
| Engine | `GravitileKit` — pure Swift SPM package, fully deterministic |
| Project | Generated by XcodeGen — run `./Tools/generate.sh` |

## Play

- **Endless** — swipe-merge under rising spawn pressure; every game has a
  cruise → tension → collapse arc (tuned by Monte-Carlo simulation).
- **Daily** — one seeded puzzle per UTC day, identical worldwide, 40 moves,
  emoji share cards, streaks with a weekly freeze.
- **Math Pop** — the learning mode: tiles carry 1–9 and merge when they
  *sum to the stage target* (Make 5 → Make 16), popping with their equation.
  Tile colors follow the classroom Cuisenaire rod system.
- **Zen** — no clock, no ending. **Sprint** — 60 moves, biggest score.

Five OKLCH-derived color themes (two of them light) live in Settings.

Monetization: free, no ads, no tracking. One non-consumable (**Gravitile
Plus**: daily archive + unlimited undo) and a tip jar.

## Development

```bash
brew install xcodegen
./Tools/generate.sh                 # generates Gravitile.xcodeproj (+ scheme patch)
open Gravitile.xcodeproj
```

- Engine tests (fast, no simulator): `swift test --package-path GravitileKit`
- App + UI tests: run the `Gravitile` scheme's tests on an iPhone simulator
- Balance harness: `swift run --package-path GravitileKit -c release BalanceSim 300 3000`
- Sound assets are synthesized: `swift Tools/gensounds.swift Gravitile/Resources/Sounds`
- App icon is generated: `python3 Tools/genicon.py …/AppIcon.appiconset/icon-1024.png`

Debug hooks (simulator): `SIMCTL_CHILD_GRAVITILE_AUTOPLAY=1` auto-plays
endless; `SIMCTL_CHILD_GRAVITILE_ROUTE=daily|stats|paywall` deep-links a
screen; launch argument `-gravitile-reset` wipes persisted state.

## Architecture

The engine emits `MoveResult` event data (slides, merges, falls per cascade
round, spawns); the app's `AnimationPlanner` replays those events as phased
SwiftUI animations. All randomness flows through a Codable `SplitMix64`, so
every game — including the global daily — is replayable and unit-testable.
